소개
=============
ReguStreaming 은 영상 기반 커뮤니케이션 서비스입니다.
##### 실시간으로 서로 영상을 보며 커뮤니케이션 할 수 있습니다, 또한 영상은 모든 접속자가 신청할 수 있고 추가된 영상은 차례대로 재생됩니다.

구현
=============
nodejs, javascript, css, html 사용합니다.
